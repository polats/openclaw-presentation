Read the `README.md` file within this skill, and explore the other available files.

If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out of `assets/` and create static HTML files for the user to view. Link the tokens from `colors_and_type.css` at the top of the HTML (either copy the file or inline-paste the `:root` block).

If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand. The UI kits in `ui_kits/` are visual-only React recreations — use them as pattern references, not as production components.

**The brand in one paragraph:** Dark ink + bone-white + emergency-orange. Grainy, CRT-glowy, square-cornered. Serif display type (Instrument Serif) + brutalist sans (Space Grotesk) + technical mono (JetBrains Mono). Lowercase poetic voice with bracketed system chatter `[ TRANSMISSION 001 ]`. AI agents are named collaborators (NEPTUNE, DECAY-7). Never emoji. Never gradients-for-decoration. Never rounded cards.

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts or production code, depending on the need.
