# Getting Started With Openclaw

An interactive presentation built with [Remotion](https://www.remotion.dev/) and [Next.js](https://nextjs.org/), deployed on GitHub Pages.

## Live

[polats.github.io/openclaw-presentation](https://polats.github.io/openclaw-presentation)

## Controls

- **Arrow keys** — navigate between slides
- **Space / Click / Tap** — pause / resume
- **Swipe left/right** — navigate slides (mobile)

## Development

```bash
npm install
npm run dev        # Next.js dev server
npx remotion studio  # Remotion Studio
```

## Deploy

Pushes to `main` auto-deploy to GitHub Pages via GitHub Actions.
